## ✨ 프로젝트 소개

<div align="center">
  <img src="https://img.icons8.com/ios-filled/500/headphones.png" width="100" />
</div>

<h3 align="center">🎧 내 손안의 음악 공간, <span style="color:#3DDC84"><b>Music APP</b></span></h3>
<br><br><br>
<p align="center">
  <i>사용자 중심의 경험, 직관적인 UI, 그리고 세련된 재생 기능을 담은 모바일 음악 앱</i>
</p>
<br><br>
**Music APP Project**는 Android Studio 기반의 **모바일 음악 재생 앱**으로,  
사용자가 **자신의 라이브러리에서 음악을 탐색, 재생, 관리**할 수 있는 기능을 제공합니다.  

### 🎯 주요 개발 목적
- 직관적인 **로컬 음악 탐색 및 재생 인터페이스** 구현
- **사용자 맞춤형 플레이리스트**와 **테마 커스터마이징** 제공
- 어두운 환경에서도 눈에 부담 없는 **다크 모드 UI 지원**
- MVVM 아키텍처와 효율적인 **백엔드 연동**을 통한 데이터 구조화
